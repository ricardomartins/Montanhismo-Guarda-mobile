# Política de Privacidade - Clube de Montanhismo da Guarda (Mobile)

Última atualização: **29 de Março de 2026**

O [Ricardo Martins](https://github.com/ricardomartins) construiu a aplicação ClubeMG Mobile como uma [aplicação Open-Source](https://github.com/ricardomartins/Montanhismo-Guarda-mobile). Este serviço é fornecido pelo *Clube de Montanhismo da Guarda* sem custos e destina-se a ser utilizado tal como está.

Esta página é usada para informar os utilizadores sobre as nossas políticas de recolha, uso e divulgação de Informações Pessoais, caso alguém decida usar o nosso Serviço.

1. **Recolha e Uso de Informações:** A aplicação ClubeMG Mobile foi desenhada para respeitar a tua privacidade. Nós não recolhemos, guardamos ou partilhamos informações pessoais identificáveis (PII) nos nossos servidores.
    - **Favoritos (Bookmarks):** Quando guardas um evento como favorito, essa informação é guardada localmente no teu próprio dispositivo.
    - **Acesso à Internet:** A aplicação necessita de ligação à internet para descarregar a lista de eventos, calendários e imagens associadas aos mesmos.
    - **Permissão de Notificações:** A aplicação solicita a permissão de notificações (POST_NOTIFICATIONS) exclusivamente para te alertar sobre eventos que tenhas marcado como favoritos. Nenhuma informação sobre os teus hábitos de uso é enviada para o exterior.
    
2. **Partilha de Dados com Terceiros:** Uma vez que não recolhemos dados pessoais, também não partilhamos, vendemos ou alugamos qualquer tipo de informação pessoal a terceiros.
3. **Segurança:** Valorizamos a tua confiança ao usar a nossa aplicação. Os dados gerados pela app (como as tuas preferências de favoritos) ficam retidos no armazenamento seguro do sistema operativo Android do teu dispositivo.
4. **Links para Outros Sites:** A aplicação pode conter links para outros sites (por exemplo, no botão "Visitar Página" dos eventos). Se clicares num link de terceiros, serás direcionado para esse site. Nota que esses sites externos não são operados por nós. Portanto, aconselhamos vivamente que revejas a Política de Privacidade desses sites.
5. **Alterações a esta Política de Privacidade:** Podemos atualizar a nossa Política de Privacidade de tempos a tempos. Assim, aconselhamos que revejas esta página periodicamente para quaisquer alterações. Iremos notificar-te de quaisquer alterações publicando a nova Política de Privacidade nesta página.
6. **Contactos:** Se tiveres alguma dúvida ou sugestão sobre a nossa Política de Privacidade, não hesites em contactar-nos através de:
    - Email: clube@montanhismo-guarda.pt
    - Website: https://ricardomartins.github.io/Montanhismo-Guarda-mobile/
